#  Capstone Project: Thread of Destiny

**University:** KBTU (Kazakh-British Technical University)  
**Department:** Computer Science  
**Course:** Capstone Design Project  
**Team:** DC (Dream Coders)  
**Semester:** Fall 2025  

---

##  Project Overview
**Thread of Destiny** is a narrative-driven adventure game focused on player choice and emotional storytelling.  
The game explores how individual decisions shape the world and the fate of its characters.

Built as part of the KBTU Capstone program, the project combines game design, programming, and narrative development.

---

##  Objectives
- Develop an interactive 3D/2D game with a rich storyline  
- Implement a dialogue system with branching choices  
- Design a user-friendly UI and immersive sound experience  
- Demonstrate gameplay mechanics and game engine capabilities (Unity/Unreal/Godot)
